## 二、实验原理

### 2.1 图像分割概述

图像分割是将图像划分为若干个互不重叠的区域的过程，使得每个区域内部具有相似的特征（如灰度、颜色、纹理等），而不同区域之间存在明显差异。

### 2.2 主要算法

#### 2.2.1 阈值分割

**Otsu阈值法**：自动确定最佳阈值，使类间方差最大化。对于灰度图像，Otsu方法通过遍历所有可能的阈值，计算前景和背景的类间方差，选择使方差最大的阈值作为最优阈值。

公式：
```
σ²(t) = ω₀(t) × ω₁(t) × [μ₀(t) - μ₁(t)]²
```
其中：
- t 为阈值
- ω₀, ω₁ 为前景和背景的像素比例
- μ₀, μ₁ 为前景和背景的平均灰度

**自适应阈值**：根据局部区域的特征动态确定阈值，适用于光照不均匀的图像。

#### 2.2.2 形态学处理

- **腐蚀（Erosion）**：去除边界像素，消除小的噪声点
- **膨胀（Dilation）**：扩展边界，填充小的孔洞
- **开运算（Opening）**：先腐蚀后膨胀，去除小物体和噪声
- **闭运算（Closing）**：先膨胀后腐蚀，填充小孔洞

#### 2.2.3 分水岭算法

分水岭算法是一种基于拓扑理论的图像分割方法，将图像看作地形表面：
1. 灰度值表示高度
2. 从局部最小值开始"注水"
3. 不同区域的水汇合处形成分水岭线
4. 分水岭线即为分割边界

改进方法：
- 使用距离变换找到细胞中心
- 标记确定的前景和背景
- 应用分水岭算法分离粘连细胞

#### 2.2.4 图像增强

**CLAHE（对比度受限自适应直方图均衡化）**：
- 将图像分成小块（如8×8）
- 对每个小块进行直方图均衡化
- 限制对比度增强的幅度，避免噪声放大
- 使用双线性插值平滑块边界

**双边滤波**：
- 同时考虑空间距离和像素值差异
- 在去噪的同时保留边缘信息
- 适合细胞图像的预处理

### 2.3 区域属性测量

对分割后的每个细胞区域，计算以下参数：

1. **长轴长度（Major Axis Length）**：拟合椭圆的长轴长度
2. **短轴长度（Minor Axis Length）**：拟合椭圆的短轴长度
3. **面积（Area）**：区域内像素总数
4. **周长（Perimeter）**：区域边界的长度
5. **圆度（Circularity）**：衡量区域接近圆形的程度
   ```
   Circularity = 4π × Area / Perimeter²
   ```
   圆度值在0到1之间，1表示完美的圆形
6. **均值（Mean Intensity）**：区域内像素灰度的平均值
7. **方差（Variance）**：区域内像素灰度的方差，反映灰度分布的离散程度

```python
def preprocess_image(self, img):
    # 双边滤波去噪，保留边缘
    denoised = cv2.bilateralFilter(img, 9, 75, 75)
    
    # CLAHE增强对比度
    clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8,8))
    enhanced = clahe.apply(denoised)
    
    return enhanced
```

```python
# Otsu阈值
_, binary1 = cv2.threshold(enhanced, 0, 255,

# 自适应阈值
binary2 = cv2.adaptiveThreshold(enhanced, 255, 
                                cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
                                cv2.THRESH_BINARY, 11, 2)

# 结合两种方法
binary = cv2.bitwise_and(binary1, binary2)
```

#### 步骤3：形态学处理

```python

kernel = morphology.disk(2)
```

#### 步骤4：分水岭分割（手动实现）

本实验**手动实现了分水岭算法**，不使用库函数，以深入理解算法原理：

```python
def manual_watershed(self, gradient, markers, mask):
    """
    手动实现分水岭算法
    
    算法原理：
    1. 将图像看作地形图，像素值代表高度
    2. 从标记点（种子点）开始，模拟水淹过程
    3. 使用优先队列按照高度从低到高处理像素
    4. 当不同区域的水相遇时，形成分水岭边界
    """
    h, w = gradient.shape
    labels = markers.copy().astype(np.int32)
    
    # 优先队列：(高度, y坐标, x坐标)
    pq = []
    
    # 初始化：将所有标记点的邻居加入优先队列
    for y in range(h):
        for x in range(w):
            if labels[y, x] > 0:  # 已标记的种子点
                # 检查8邻域
                for dy in [-1, 0, 1]:
                    for dx in [-1, 0, 1]:
                        if dy == 0 and dx == 0:
                            continue
                        ny, nx = y + dy, x + dx
                        if 0 <= ny < h and 0 <= nx < w:
                            if mask[ny, nx] > 0 and labels[ny, nx] == 0:
                                heapq.heappush(pq, (int(gradient[ny, nx]), ny, nx))
    
    # 分水岭主循环：按高度从低到高处理像素
    while pq:
        height, y, x = heapq.heappop(pq)
        
        if labels[y, x] != 0:
            continue
        
        # 检查邻域，找到该像素应该属于的区域
        neighbor_labels = []
        for dy in [-1, 0, 1]:
            for dx in [-1, 0, 1]:
                if dy == 0 and dx == 0:
                    continue
                ny, nx = y + dy, x + dx
                if 0 <= ny < h and 0 <= nx < w:
                    if labels[ny, nx] > 0:
                        neighbor_labels.append(labels[ny, nx])
        
        # 如果邻域只有一个标签，分配该标签
        if len(set(neighbor_labels)) == 1:
            labels[y, x] = neighbor_labels[0]
            # 将未访问的邻居加入队列
            for dy in [-1, 0, 1]:
                for dx in [-1, 0, 1]:
                    if dy == 0 and dx == 0:
                        continue
                    ny, nx = y + dy, x + dx
                    if 0 <= ny < h and 0 <= nx < w:
                        if mask[ny, nx] > 0 and labels[ny, nx] == 0:
                            heapq.heappush(pq, (int(gradient[ny, nx]), ny, nx))
        # 如果邻域有多个不同标签，标记为分水岭边界
        elif len(set(neighbor_labels)) > 1:
            labels[y, x] = -1
    
    return labels
```

**算法特点**：
- 使用优先队列（heapq）实现高效的像素排序
- 按梯度高度从低到高处理，模拟真实的水淹过程
- 当不同区域相遇时标记为分水岭边界（-1）
- 完全不依赖任何图像处理库的分水岭函数

```python
# 修复前（错误）

```

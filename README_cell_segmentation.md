# 细胞分割程序使用说明

## 功能介绍

本程序实现了基于图像处理的细胞分割功能，可以：
1. 自动分割图像中的细胞
2. 交互式显示分割结果
3. 点击任意细胞查看其详细参数

## 算法原理

程序采用以下图像处理技术：

### 1. 预处理
- **高斯滤波**：去除图像噪声
- **直方图均衡化**：增强图像对比度

### 2. 分割算法
- **Otsu自动阈值**：自动确定最佳分割阈值
- **形态学操作**：去除小噪点，平滑边界
- **距离变换**：找到细胞中心区域
- **分水岭算法**：分离粘连的细胞

### 3. 参数计算
对每个分割出的细胞计算以下参数：
- **长轴长度**：细胞的最长直径
- **短轴长度**：细胞的最短直径
- **圆度**：衡量细胞形状接近圆形的程度（公式：4π×面积/周长²）
- **面积**：细胞占据的像素数量
- **周长**：细胞边界的长度
- **均值**：细胞内像素灰度的平均值
- **方差**：细胞内像素灰度的方差

## 使用方法

### 基本用法

```bash
python3 cell_segmentation.py Picture1.png
```

### 程序运行流程

1. **加载图像**：读取指定的细胞图像
2. **自动分割**：使用分水岭算法分割细胞
3. **保存结果**：生成 `cell_segmentation_result.png` 文件
4. **交互界面**：弹出窗口显示分割结果

### 交互操作

在弹出的窗口中：
- **点击任意细胞**：显示该细胞的详细参数
- 参数会以黄色标注框显示在细胞旁边
- 同时在控制台打印详细信息

## 输出文件

### cell_segmentation_result.png
包含4个子图的综合结果：
1. **原始图像**：输入的细胞图像
2. **灰度图像**：转换后的灰度版本
3. **分割结果**：用不同颜色标记的细胞区域
4. **叠加显示**：在原图上叠加分割结果

## 参数说明

### 圆度 (Circularity)
- 公式：`4π × 面积 / 周长²`
- 取值范围：0 到 1
- 完美圆形的圆度为 1
- 越接近 1 表示细胞越圆
- 越接近 0 表示细胞越不规则

### 长轴/短轴
- 通过拟合椭圆得到
- 长轴：椭圆的最长直径
- 短轴：椭圆的最短直径
- 可用于判断细胞的伸长程度

## 常见问题

### Q1: 分割结果过度分割（一个细胞被分成多个）
**解决方法**：
- 增大分水岭阈值系数（如从 0.3 改为 0.4）
- 减少形态学操作的迭代次数

### Q2: 粘连细胞未能分离
**解决方法**：
- 减小分水岭阈值系数（如从 0.3 改为 0.2）
- 增加膨胀操作的迭代次数

### Q3: 检测到很多小噪点
**解决方法**：
- 增大高斯滤波核
- 增加形态学开运算的迭代次数
- 添加面积过滤（过滤掉面积过小的区域）

### Q4: 中文显示为方框
**解决方法**：
程序已自动处理，使用英文标签显示参数

## 技术细节

### 分水岭算法原理
1. 将图像看作地形图，灰度值代表高度
2. 从局部最小值开始"注水"
3. 不同区域的水相遇处形成分水岭
4. 分水岭即为细胞边界

### 距离变换
- 计算每个前景像素到最近背景像素的距离
- 距离大的点位于细胞中心
- 用于确定细胞的"种子点"
